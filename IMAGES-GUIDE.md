## How to Replace Images

### For Static HTML Version:
1. Simply replace the files in the `assets/images/` folder
2. Keep the same filenames or update references in `index.html`

### For WordPress Theme:
1. Upload new images through WordPress Media Library
2. Or replace files directly in the theme folder
3. Update image paths in template files if needed

## Image Optimization Tips

1. **Compress images** before uploading (use TinyPNG, ImageOptim, etc.)
2. **Use WebP format** for better performance (with fallbacks)
3. **Lazy load** images below the fold
4. **Use appropriate dimensions** - don't upload 4K images for 200px displays
5. **Add alt text** for accessibility and SEO

## Recommended Image Sizes

- **Logo**: 200x50px (SVG preferred)
- **Hero Image**: 1200x800px
- **About Image**: 800x800px
- **Website Mockup**: 1200x900px
- **Company Logos**: 200x80px (transparent background)
